This is how it works

BIP32 is a proposal to improve the bitcoin ecosystem that established a standard way to generate multiple private keys from a seed of random bits and a path. This example shows how to use the bitcore API to obtain deterministic, hierarchically-derived private and public keys from a seed value.

Read more in the bitcore developer guide.
